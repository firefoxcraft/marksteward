What is HTML?
A HTML or Hyper Text Markup Language is a language that most programmers use to create and design a website.
HTML requires Critical,Technical and mostly, Artistic skills in order to create an attractive website that will catch the
consumer's eyes. However, even though most programmers use this language, the HTML is not a programming language.
It is meant to design out of something and not to program out of something. Learning HTML sytaxes doesn't make you a
Programmer.
